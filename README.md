# Help-Protect-the-Great-Barrier-Reef
## Goal
The goal of this mini project was to accurately identify starfish in real-time by building an object detection model trained on underwater videos of coral reefs.
## Dataset
The dataset came from [here](https://www.kaggle.com/competitions/tensorflow-great-barrier-reef/data).
## Model
The model I've used was YoloX-S with F2 score of 0.59.
